# Instagram-Project# 🌐 Social Media App (MERN Stack)

## 📌 Project Overview

This is a basic Social Media Application built using the MERN stack (MongoDB, Express.js, React.js, Node.js).
Users can register, login, and interact with posts.

---

## 🚀 Features

* 👤 User Registration & Login
* 🔐 Authentication system
* 📝 Create Posts
* 📄 View All Posts
* ❤️ Like Posts
* 👥 Follow Users (in progress)

---

## 🛠️ Tech Stack

**Frontend:**

* React.js
* React Router
* SCSS

**Backend:**

* Node.js
* Express.js
* MongoDB (Mongoose)

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/your-repo-name.git
```

### 2️⃣ Install dependencies

**Client:**

```
cd client
npm install
```

**Server:**

```
cd server
npm install
```

---

### 3️⃣ Run the project

**Start Backend:**

```
npm start
```

**Start Frontend:**

```
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file in server folder:

```
PORT=5000
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret_key
```
