# Instagram-Project# 🌐 Social Media App (MERN Stack)

## 📌 Project Overview

This is a basic Social Media Application built using the MERN stack (MongoDB, Express.js, React.js, Node.js).
Users can register, login, and interact with posts.

---

## 🚀 Features

* 👤 User Registration & Login
* 🔐 Authentication system
* 📝 Create Posts
* 📄 View All Posts
* ❤️ Like Posts
* 👥 Follow Users (in progress)

---

## 🛠️ Tech Stack

**Frontend:**

* React.js
* React Router
* SCSS

**Backend:**

* Node.js
* Express.js
* MongoDB (Mongoose)
